# Release History

## 1.0.0-beta.1 (2026-04-29)

### Features Added

- Initial beta release of Azure.Provisioning.Logic with support for declarative Azure Logic Apps resource provisioning.
- Added support for the following resources: `IntegrationAccount`, `IntegrationAccountAgreement`, `IntegrationAccountAssemblyDefinition`, `IntegrationAccountBatchConfiguration`, `IntegrationAccountCertificate`, `IntegrationAccountMap`, `IntegrationAccountPartner`, `IntegrationAccountSchema`, `IntegrationAccountSession`, `IntegrationServiceEnvironment`, `IntegrationServiceEnvironmentManagedApi`, `LogicWorkflow`.
