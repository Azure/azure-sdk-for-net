# Vector Search Using Image Vector Queries

This sample demonstrates how to create a vector index with image vectorization support and perform vector searches using image inputs. Azure AI Search supports two types of image-based vector queries:

- **`VectorizableImageUrlQuery`** — provide a publicly accessible URL to an image; the configured vectorizer fetches and vectorizes it at query time.
- **`VectorizableImageBinaryQuery`** — provide the image directly as a base64-encoded string; the configured vectorizer vectorizes it at query time.

Both query types require the target vector field to be backed by a vectorizer that supports image inputs, such as the [Azure AI Vision Vectorize API](https://learn.microsoft.com/azure/ai-services/computer-vision/how-to/image-retrieval) or a custom Web API vectorizer.

Please refer to the [vector search documentation](https://learn.microsoft.com/azure/search/vector-search-overview) to learn more.

## Create a Vector Index with an Image Vectorizer

Create an index with a `VectorSearchField` and configure a vectorizer (here a `WebApiVectorizer` pointing to an Azure AI Vision endpoint). The vectorizer is referenced by name from the vector search profile.

```C# Snippet:Azure_Search_Documents_Tests_Samples_Sample07_Image_Vector_Search_Index
string vectorSearchProfileName = "my-image-vector-profile";
string vectorSearchHnswConfig = "my-hnsw-vector-config";
string vectorizerName = "my-ai-vision-vectorizer";
int modelDimensions = 1024;

string indexName = "hotel";
SearchIndex searchIndex = new(indexName)
{
    Fields =
    {
        new SimpleField("HotelId", SearchFieldDataType.String) { IsKey = true, IsFilterable = true, IsSortable = true, IsFacetable = true },
        new SearchableField("HotelName") { IsFilterable = true, IsSortable = true },
        new SearchableField("Description") { IsFilterable = true },
        new VectorSearchField("DescriptionImageVector", modelDimensions, vectorSearchProfileName),
    },
    VectorSearch = new()
    {
        Profiles =
        {
            new VectorSearchProfile(vectorSearchProfileName, vectorSearchHnswConfig)
            {
                VectorizerName = vectorizerName
            }
        },
        Algorithms =
        {
            new HnswAlgorithmConfiguration(vectorSearchHnswConfig)
        },
        Vectorizers =
        {
            new WebApiVectorizer(vectorizerName)
            {
                Parameters = new WebApiVectorizerParameters
                {
                    Uri = new Uri(Environment.GetEnvironmentVariable("AIVISION_ENDPOINT")),
                    HttpHeaders =
                    {
                        ["api-key"] = Environment.GetEnvironmentVariable("AIVISION_KEY")
                    }
                }
            }
        }
    },
};
```

After creating the index definition, instantiate `SearchIndexClient` and call `CreateIndexAsync`:

```C# Snippet:Azure_Search_Documents_Tests_Samples_Sample07_Image_Vector_Search_Create_Index
Uri endpoint = new(Environment.GetEnvironmentVariable("SEARCH_ENDPOINT"));
string key = Environment.GetEnvironmentVariable("SEARCH_API_KEY");
AzureKeyCredential credential = new(key);

SearchIndexClient indexClient = new(endpoint, credential);
await indexClient.CreateIndexAsync(searchIndex);
```

## Add Documents to the Index

Define a model type that includes the image vector field:

```C# Snippet:Azure_Search_Documents_Tests_Samples_Sample07_Image_Vector_Search_Hotel_Model
public class ImageHotel
{
    public string HotelId { get; set; }
    public string HotelName { get; set; }
    public string Description { get; set; }
    public ReadOnlyMemory<float> DescriptionImageVector { get; set; }
}
```

Upload documents using a `SearchClient`. The `DescriptionImageVector` field should contain image embeddings generated by calling your image vectorization service:

```C# Snippet:Azure_Search_Documents_Tests_Samples_Sample07_Image_Vector_Search_Upload_Documents
SearchClient searchClient = new(endpoint, indexName, credential);
ImageHotel[] hotelDocuments = GetHotelDocuments();
await searchClient.IndexDocumentsAsync(IndexDocumentsBatch.Upload(hotelDocuments));
```

## Search Using an Image URL

Use `VectorizableImageUrlQuery` to search by providing a URL to a publicly accessible image. The service's configured vectorizer fetches the image and converts it to a vector at query time.

```C# Snippet:Azure_Search_Documents_Tests_Samples_Sample07_Vector_Search_Using_ImageUrl
SearchResults<ImageHotel> response = await searchClient.SearchAsync<ImageHotel>(
    new SearchOptions
    {
        VectorSearch = new()
        {
            Queries =
            {
                new VectorizableImageUrlQuery
                {
                    Url = new Uri("https://example.com/hotel-exterior.jpg"),
                    KNearestNeighborsCount = 3,
                    Fields = { "DescriptionImageVector" }
                }
            }
        }
    });

int count = 0;
Console.WriteLine("Image URL Vector Search Results:");
await foreach (SearchResult<ImageHotel> result in response.GetResultsAsync())
{
    count++;
    ImageHotel doc = result.Document;
    Console.WriteLine($"{doc.HotelId}: {doc.HotelName}");
}
Console.WriteLine($"Total number of search results: {count}");
```

## Search Using a Base64-Encoded Image

Use `VectorizableImageBinaryQuery` to search by providing raw image bytes as a base64-encoded string. This is useful when the image is not publicly accessible via URL.

```C# Snippet:Azure_Search_Documents_Tests_Samples_Sample07_Vector_Search_Using_ImageBinary
// Read image bytes and convert to base64
byte[] imageBytes = System.IO.File.ReadAllBytes("path/to/your/image.jpg");
string base64Image = Convert.ToBase64String(imageBytes);

SearchResults<ImageHotel> response = await searchClient.SearchAsync<ImageHotel>(
    new SearchOptions
    {
        VectorSearch = new()
        {
            Queries =
            {
                new VectorizableImageBinaryQuery
                {
                    Base64Image = base64Image,
                    KNearestNeighborsCount = 3,
                    Fields = { "DescriptionImageVector" }
                }
            }
        }
    });

int count = 0;
Console.WriteLine("Image Binary Vector Search Results:");
await foreach (SearchResult<ImageHotel> result in response.GetResultsAsync())
{
    count++;
    ImageHotel doc = result.Document;
    Console.WriteLine($"{doc.HotelId}: {doc.HotelName}");
}
Console.WriteLine($"Total number of search results: {count}");
```
