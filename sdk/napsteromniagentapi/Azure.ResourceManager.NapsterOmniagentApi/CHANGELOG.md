# Release History

## 1.0.0-beta.1 (2026-06-29)

### Features Added

- Initial preview release of Azure.ResourceManager.NapsterOmniagentApi management library.
- Support for managing Napster CompanionAPI organization resources.
- Operations: Create, Read, Update, Delete, List organizations.
- SaaS linking: Link and retrieve linked SaaS resources.
- SaaS activation: Activate and validate SaaS resources.
