# The plugin to set experimental attributes

This plugin depends on three CSV files, generated by the `typespec-extension-exporter` package, which builds a table of experimental fields and classes based on `@extension("x-ms-foundry-meta", ...)` decorator.

To update the table after the typespec change, please run the following commands in the project folder.

```bash
npx tsp-client sync
cd .\TempTypeSpecFiles\
npm install --no-save typespec-extension-exporter
# For Azure.AI.Projects.Agents
npx tsp compile sdk-csharp-azure-ai-projects-agents\client.tsp --emit typespec-extension-exporter
# For Azure.AI.Extensions.OpenAI
npx tsp compile sdk-csharp-azure-ai-extensions-openai\client.tsp --emit typespec-extension-exporter
# For Azure.AI.Projects
npx tsp compile sdk-csharp-azure-ai-projects\client.tsp --emit typespec-extension-exporter
```
