Breaking Changes
================
12.0.0 (2019-11)
--------------------------
- Make ContentLanguage and ContentEncoding properties strings everywhere as opposed to being arrays or IEnumerable<string> in some places.
- Update BlobContainerClient.GetProperties to return Response<BlobContainerProperties> instead of Response<BlobContainerItem>.
- Move Metadata from BlobContainerItem to BlobContainerProperties.
- Make Metadata property null instead of empty dictionary if Metadata not requested in GetBlobs/GetContainers.
- Rename LeaseClient to LeaseBlobClient.
- Rename Block to BlobBlock.
- Rename the parallel download overloads to DownloadTo.
- Remove default constructors for BlobItem, BlobContainerItem, BlockList, and BlobServiceStatistics.
- Rename Metrics.IncludeAPIs to Metrics.IncludeApis.
- Rename SignedIdentifier to BlobSignedIdentifier.
- Rename RetentionPolicy to BlobRetentionPolicy.
- Rename AccessPolicy to BlobAccessPolicy.
- Move ContentType from BlobDownloadProperties to BlobDownloadInfo.
- Rename BlobDownloadProperties to BlobDownloadDetails.
- Change return type of BlobClient.StartCopyFromUri to be CopyFromUriOperation.
- Rename all instances of Expiry to ExpiresOn, and Starts to StartsOn
- Update the service version specified in BlobClientOptions to be The 2019-02-02. If anything else is specified in the constructor call, a version not supported exception is thrown.
- Remove the "Blob" suffix from BlobType enum values.
- Update SkuName enum values acronyms to be PascalCase rather than all CAPS.
- Rename DeleteSnapshotsOption.Include to DeleteSnapshotsOption.IncludeSnapshots
- Rename DeleteSnapshotsOption.Only to DeleteSnapshotsOption.OnlySnapshots
- Rename parameters blobAccessConditions and httpAccessConditions to accessConditions.
- Rename parameters blobHttpHeaders to httpHeaders.
- Remove WithCustomerProvidedKey constructors in favor of adding the CustomerProvidedKey as optional parameter to other existing constructors
- Rename Tid/SignedTid to TenantId/SignedTenantId
- Update BlobAccountSasPermissions/BlobContainerSasPermissions/BlobSasPermissions/SnapshotSasPermissions to be Flags Enums rather than structs
- Update BlobSasBuilder to have SetPermissions methods with overloads for the various Permissions enums. Make the string Permissions property read only

12.0.0-preview.4 (2019-10)
--------------------------
- Removing Uri property from BlobUriBuilder and replacing with a ToUri method
  that returns the Uri.
- Change BlobItem.Deleted and BlobItemProperties.AccessTierInferred from bool?
  to bool
- Replace GetBlobsOptions with BlobTraits and BlobStates enums/parameters
- Replace GetContainersOptions with BlobContainerTraits
- Change BlockListType to a BlockListTypes flags enum
- Changed ArchiveStatus and RehydratePriority into enums
- Added Blob prefix to all APIs starting with Container
- Added Blob prefix to all the BlobServiceClient.GetProperties return types
- Changed PageInfo, PageRange, and AccountInfo to structs
- Renamed SetTier to SetAccessTier
- Made Pipeline property internal
- Changed LeaseClient.Break to use a TimeSpan
- Removed public constructors from output only Models

12.0.0-preview.3 (2019-09)
--------------------------
- Changing from Nullable<PublicAccessType> to just PublicAccessType in all
  relevant APIs. Use PublicAccessType.None instead of null.
- Changed from int to TimeSpan for Lease durations.

12.0.0-preview.2 (2019-08)
--------------------------
- Removed CopyFromUriOperation.  Use Operation<long> instead.

12.0.0-preview.1 (2019-07)
--------------------------
- New Azure.Storage.Blobs client library.  For more information, please visit:
  https://aka.ms/azure-sdk-preview1-net.
