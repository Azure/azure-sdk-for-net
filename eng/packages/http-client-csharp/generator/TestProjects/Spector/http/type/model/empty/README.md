# Type.Model.Empty client library for .NET

Type.Model.Empty is a client library for developing .NET applications with rich experience.

## Getting started

### Install the package

Install the client library for .NET with [NuGet](https://www.nuget.org/):

```dotnetcli
dotnet add package Type.Model.Empty --prerelease
```

### Prerequisites

- You must have a [Microsoft Azure subscription](https://azure.microsoft.com/free/dotnet/).
